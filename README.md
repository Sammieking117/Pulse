# Pulse (Spotify-inspired)

A responsive music discovery dashboard inspired by modern streaming platforms
such as Spotify.
This project focuses on frontend architecture, routing, reusable components,
and responsive UI design.

Note: This is not a full spotify clone and does not include music playback.

## Project Purpose

The purpose of this project is to demonstrate:
- Component-based architecture with react
- Client-side routing and dynamic pages
- Data-driven UI rendering
- Responsive layouts across devices
- Clean and scalable project structure

The project scope is intentionally limited to focus on frontend fundamentals.
